# Admin Manual

Authored. Chapters `01`..`10` are the source; `meta.yaml` sets the cover and version. Build with `python3 build.py admin-manual`. Covers the in-app Admin Console (Users, self-enrolment, editor access, galaxies/clusters, global settings, federation, backups/snapshots, maintenance) as of Telaris v6.12.28. Screenshots referenced under `assets/images/admin-manual/` are captured by the screenshot pipeline.
