# Manual de administração

Redigido. Os capítulos `01`..`10` são a fonte; `meta.yaml` define a capa e a versão. Construa com `python3 build.py admin-manual-pt`. Cobre o Console de administração dentro do aplicativo (Usuárias, auto-inscrição, acesso de edição, galáxias/aglomerados, configurações globais, federação, backups/snapshots, manutenção) no Telaris v6.12.28. As capturas de tela referenciadas em `assets/images/admin-manual-pt/` são geradas pelo pipeline de capturas.
