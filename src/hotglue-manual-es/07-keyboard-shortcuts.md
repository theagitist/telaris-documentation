# Atajos de teclado

Todos los atajos del editor de Hotglue, en un solo lugar. Funcionan en el modo de edición.

## Menús

| Atajo | Qué hace |
| --- | --- |
| Un solo clic en el fondo | Abre el menú NEW (añadir contenido) |
| <kbd>alt</kbd> + <kbd>o</kbd> | Abre el menú NEW |
| Doble clic en el fondo | Abre el menú PAGE (herramientas de página completa) |
| <kbd>alt</kbd> + <kbd>p</kbd> | Abre el menú PAGE |

## Seleccionar

| Atajo | Qué hace |
| --- | --- |
| Clic | Selecciona un objeto |
| <kbd>shift</kbd> + clic | Añade o quita un objeto de la selección |
| Clic en el fondo | Deselecciona todo |
| <kbd>ctrl</kbd> + <kbd>A</kbd> | Selecciona todos los objetos |
| <kbd>ctrl</kbd> + <kbd>D</kbd> | No selecciona ninguno |
| <kbd>ctrl</kbd> + <kbd>I</kbd> | Invierte la selección |
| <kbd>tab</kbd> | Avanza al siguiente objeto |

## Mover y redimensionar

| Atajo | Qué hace |
| --- | --- |
| Arrastrar | Mueve la selección |
| <kbd>shift</kbd> + arrastrar | Mueve solo en un eje |
| <kbd>ctrl</kbd> + arrastrar | Mueve ignorando la cuadrícula de ajuste |
| Teclas de flecha | Empuja la selección un píxel |
| <kbd>shift</kbd> + flecha | Mueve un paso de cuadrícula |
| Arrastrar el tirador de tamaño | Redimensiona el objeto |
| <kbd>ctrl</kbd> + redimensionar | Redimensiona ignorando la cuadrícula de ajuste |

## Organizar y eliminar

| Atajo | Qué hace |
| --- | --- |
| <kbd>shift</kbd> + <kbd>page up</kbd> | Lleva la selección al frente |
| <kbd>shift</kbd> + <kbd>page down</kbd> | Envía la selección al fondo |
| <kbd>delete</kbd> | Elimina la selección |

## Historial

| Atajo | Qué hace |
| --- | --- |
| <kbd>ctrl</kbd> + <kbd>z</kbd> | Se ofrece a abrir las revisiones de la página (no hay un deshacer sencillo) |

> [!note] En Windows y Linux usa Ctrl; en una Mac las mismas teclas llevan la etiqueta Control. Los atajos de Hotglue son los mismos en todas las plataformas.
