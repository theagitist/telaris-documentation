# Gerenciando páginas

A visão **Conteúdo hotglue** é onde você acompanha as suas páginas Hotglue: criá-las, nomeá-las, anexá-las a buracos de minhoca e arrumá-las. Este capítulo percorre essa visão.

## A lista

A visão mostra uma tabela das páginas Hotglue que você pode alcançar, com uma contagem no cabeçalho. As colunas são:

- uma caixa de seleção para selecionar a linha,
- **Título**,
- **Buraco de minhoca atribuído** (o buraco de minhoca em que esta página é mostrada, ou em branco se não houver),
- **Atualizado**,
- **Ações**.

![A lista de Conteúdo hotglue com três páginas: uma atribuída a um buraco de minhoca, uma sem atribuição e uma segunda página atribuída, cada uma com uma data de Atualizado e um menu de ações](assets/images/hotglue-manual-pt/02-hotglue-content-list.png)

Clique no título de uma página para abri-la na sobreposição do editor. Clique no cabeçalho da coluna **Título**, **Buraco de minhoca atribuído** ou **Atualizado** para ordenar por aquela coluna; clique de novo para inverter a ordem. Uma pequena seta marca a coluna pela qual você ordenou.

## Criando e nomeando uma página

Clique em **Nova página** para criar uma. Ela abre imediatamente no editor como uma página sem título e sem atribuição, pronta para compor.

Para renomear uma página, abra-a e edite o campo **Nome da página** no topo da sobreposição. O novo nome é salvo sozinho quando você clica fora; uma pequena nota de estado confirma.

## Atribuindo uma página a um buraco de minhoca

Uma página Hotglue só aparece para quem visita depois de estar anexada a um buraco de minhoca. Dentro da sobreposição do editor, use o controle **Buraco de minhoca atribuído**:

- É uma lista pesquisável. Comece a digitar o nome de um buraco de minhoca para reduzir as opções. Cada opção é mostrada como o nome do buraco de minhoca seguido da sua galáxia, para que você consiga distinguir buracos de minhoca com nomes parecidos.
- Escolha um buraco de minhoca para atribuir a página a ele. Escolha **Sem atribuição** para desanexar a página de novo (o buraco de minhoca então volta à sua mídia clássica; a página em si não é excluída).

Se o buraco de minhoca que você escolher já mostrar uma página Hotglue diferente, o Hotglue pergunta antes: "Substituir? Este buraco de minhoca já mostra uma página hotglue. A página que ele mostra agora ficará sem atribuição (não é excluída)." Assim você nunca perde uma página ao reatribuir um buraco de minhoca; a página deslocada simplesmente fica sem atribuição.

> [!note] Você precisa de acesso de edição à galáxia de destino
> Atribuir ou desanexar uma página exige um assento de leitura e escrita na galáxia do buraco de minhoca. Um assento somente leitura deixa você ver as páginas, mas não alterar a que elas estão anexadas.

## Duplicando uma página

A ação **Duplicar** no menu de uma linha faz uma cópia de uma página (o seu título ganha um sufixo "(cópia)"). A cópia é sempre criada **sem atribuição**, então duplicar nunca mexe no buraco de minhoca do original. Logo após duplicar, o Hotglue se oferece para atribuir a nova cópia a algum lugar.

## Vendo uma página

O menu de ações de uma linha dá três formas de olhar uma página:

- **Ver no navegador** abre a página Hotglue pura numa nova aba, exatamente como quem visita a veria, e copia o seu endereço para a área de transferência. Isso funciona para qualquer página, atribuída ou não.
- **Copiar URL direta** copia esse mesmo endereço para a área de transferência sem abrir uma aba. Use quando você só quer o link para colar em algum lugar.
- **Ver no buraco de minhoca** abre uma prévia da página dentro do seu buraco de minhoca, na mesma janela. Disponível apenas para páginas atribuídas.
- **Ver na galáxia** abre o visualizador tridimensional ao vivo da galáxia focado no buraco de minhoca, numa nova janela. Disponível apenas para páginas atribuídas.

## Revisando versões anteriores

Enquanto uma página está aberta no editor, o botão **Revisões** abre o seu histórico, para que você possa revisar versões anteriores e se recuperar de uma alteração que você não quis manter. Veja *Salvamento e histórico* em *A tela* para entender como o histórico funciona.

## Buscando e filtrando

- A caixa **Buscar** filtra a lista enquanto você digita, correspondendo ao título da página, ao nome do buraco de minhoca e ao nome da galáxia.
- O seletor de galáxia no cabeçalho do editor também serve de filtro: escolha uma galáxia para mostrar apenas as páginas Hotglue cujos buracos de minhoca ficam lá, ou deixe em todas as galáxias para ver tudo.
- **Páginas sem atribuição aparecem em qualquer filtro de galáxia.** Uma página não anexada a nenhum buraco de minhoca não tem galáxia própria, então permanece visível para qualquer galáxia que você filtre. Uma página que você acabou de fazer não vai "sumir" porque você trocou o seletor de galáxia; ela está simplesmente sem atribuição até você anexá-la.

## Trabalhando em lote

Marque as caixas de seleção (ou a caixa selecionar tudo no cabeçalho) para agir sobre várias páginas de uma vez. Aparece uma barra mostrando quantas estão selecionadas, com:

- **Limpar seleção**,
- **Desatribuir selecionadas** (desanexar as páginas escolhidas dos seus buracos de minhoca),
- **Excluir selecionadas**.

## Excluindo uma página

Exclua uma única página pelo menu de ações da sua linha, ou várias de uma vez com **Excluir selecionadas**. De qualquer forma, o Hotglue confirma antes, porque a exclusão é permanente: "Excluir esta página hotglue? Isso remove o conteúdo permanentemente. Se estiver atribuída a um buraco de minhoca, esse buraco volta para a mídia clássica." Excluir uma página remove o seu conteúdo de vez; qualquer buraco de minhoca a que ela estava anexada simplesmente volta a mostrar a mídia clássica.
