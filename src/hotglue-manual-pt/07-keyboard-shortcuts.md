# Atalhos de teclado

Todos os atalhos do editor Hotglue, num só lugar. Eles funcionam no modo de edição.

## Menus

| Atalho | O que faz |
| --- | --- |
| Clicar uma vez no fundo | Abre o menu NEW (adicionar conteúdo) |
| <kbd>alt</kbd> + <kbd>o</kbd> | Abre o menu NEW |
| Clicar duas vezes no fundo | Abre o menu PAGE (ferramentas de página inteira) |
| <kbd>alt</kbd> + <kbd>p</kbd> | Abre o menu PAGE |

## Selecionando

| Atalho | O que faz |
| --- | --- |
| Clicar | Selecionar um objeto |
| <kbd>shift</kbd> + clicar | Adicionar ou remover um objeto da seleção |
| Clicar no fundo | Desmarcar tudo |
| <kbd>ctrl</kbd> + <kbd>A</kbd> | Selecionar todos os objetos |
| <kbd>ctrl</kbd> + <kbd>D</kbd> | Não selecionar nenhum |
| <kbd>ctrl</kbd> + <kbd>I</kbd> | Inverter a seleção |
| <kbd>tab</kbd> | Avançar para o próximo objeto |

## Movendo e redimensionando

| Atalho | O que faz |
| --- | --- |
| Arrastar | Mover a seleção |
| <kbd>shift</kbd> + arrastar | Mover apenas em um eixo |
| <kbd>ctrl</kbd> + arrastar | Mover ignorando a grade de encaixe |
| Teclas de seta | Deslocar a seleção um pixel |
| <kbd>shift</kbd> + seta | Mover por um passo da grade |
| Arrastar a alça de redimensionamento | Redimensionar o objeto |
| <kbd>ctrl</kbd> + redimensionar | Redimensionar ignorando a grade de encaixe |

## Organizando e removendo

| Atalho | O que faz |
| --- | --- |
| <kbd>shift</kbd> + <kbd>page up</kbd> | Trazer a seleção para a frente |
| <kbd>shift</kbd> + <kbd>page down</kbd> | Enviar a seleção para o fundo |
| <kbd>delete</kbd> | Excluir a seleção |

## Histórico

| Atalho | O que faz |
| --- | --- |
| <kbd>ctrl</kbd> + <kbd>z</kbd> | Oferece abrir as revisões da página (não há desfazer comum) |

> [!note] No Windows e no Linux use Ctrl; num Mac as mesmas teclas são rotuladas como Control. Os atalhos do Hotglue são os mesmos em todas as plataformas.
