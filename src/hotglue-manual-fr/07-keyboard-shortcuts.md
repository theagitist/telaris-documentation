# Raccourcis clavier

Tous les raccourcis de l'éditeur Hotglue, au même endroit. Ils fonctionnent en mode édition.

## Menus

| Raccourci | Ce qu'il fait |
| --- | --- |
| Clic simple sur l'arrière-plan | Ouvrir le menu NEW (ajouter du contenu) |
| <kbd>alt</kbd> + <kbd>o</kbd> | Ouvrir le menu NEW |
| Double-clic sur l'arrière-plan | Ouvrir le menu PAGE (outils pleine page) |
| <kbd>alt</kbd> + <kbd>p</kbd> | Ouvrir le menu PAGE |

## Sélection

| Raccourci | Ce qu'il fait |
| --- | --- |
| Clic | Sélectionner un objet |
| <kbd>shift</kbd> + clic | Ajouter ou retirer un objet de la sélection |
| Clic sur l'arrière-plan | Tout désélectionner |
| <kbd>ctrl</kbd> + <kbd>A</kbd> | Sélectionner tous les objets |
| <kbd>ctrl</kbd> + <kbd>D</kbd> | Ne rien sélectionner |
| <kbd>ctrl</kbd> + <kbd>I</kbd> | Inverser la sélection |
| <kbd>tab</kbd> | Passer à l'objet suivant |

## Déplacer et redimensionner

| Raccourci | Ce qu'il fait |
| --- | --- |
| Glisser | Déplacer la sélection |
| <kbd>shift</kbd> + glisser | Déplacer sur un seul axe |
| <kbd>ctrl</kbd> + glisser | Déplacer en ignorant la grille d'aimantation |
| Touches fléchées | Décaler la sélection d'un pixel |
| <kbd>shift</kbd> + flèche | Déplacer d'un pas de grille |
| Glisser la poignée de redimensionnement | Redimensionner l'objet |
| <kbd>ctrl</kbd> + redimensionner | Redimensionner en ignorant la grille d'aimantation |

## Arranger et retirer

| Raccourci | Ce qu'il fait |
| --- | --- |
| <kbd>shift</kbd> + <kbd>page up</kbd> | Amener la sélection à l'avant |
| <kbd>shift</kbd> + <kbd>page down</kbd> | Envoyer la sélection à l'arrière |
| <kbd>delete</kbd> | Supprimer la sélection |

## Historique

| Raccourci | Ce qu'il fait |
| --- | --- |
| <kbd>ctrl</kbd> + <kbd>z</kbd> | Proposer d'ouvrir les révisions de la page (il n'y a pas d'annulation simple) |

> [!note] Sous Windows et Linux, utilise Ctrl ; sur un Mac, les mêmes touches sont étiquetées Control. Les raccourcis de Hotglue sont les mêmes sur toutes les plateformes.
