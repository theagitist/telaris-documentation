# Glossaire

**Page.** Une seule toile Hotglue : la surface sur laquelle tu arranges des objets. Une page peut être rattachée à un trou de ver pour que celui-ci l'affiche.

**Objet.** Tout ce que tu poses sur une page : une image, une boîte de texte, une vidéo, un dessin. Chaque objet peut être déplacé, redimensionné, superposé, verrouillé, transformé en lien et supprimé.

**Menu NEW.** Le menu pour ajouter des objets à une page. Ouvre-le en cliquant une fois sur l'arrière-plan vide, ou avec <kbd>alt</kbd> + <kbd>o</kbd>.

**Menu PAGE.** Le menu pour les réglages pleine page (arrière-plan, grille). Ouvre-le en double-cliquant sur l'arrière-plan vide, ou avec <kbd>alt</kbd> + <kbd>p</kbd>.

**Menu contextuel.** L'éventail de petites icônes d'outils qui apparaît autour d'un objet quand tu le sélectionnes. Ses outils dépendent du type de l'objet.

**Grille.** L'espacement invisible auquel les objets s'aimantent quand tu les déplaces et les redimensionnes. Ajustable et masquable depuis le menu PAGE ; maintiens <kbd>ctrl</kbd> pour l'ignorer le temps d'un déplacement.

**Révisions.** L'historique d'une page. Ouvre-le avec le bouton **Révisions** dans l'éditeur. Hotglue prend un instantané d'une page environ une fois par heure pendant qu'elle est éditée et conserve les instantanés pendant sept jours. Les révisions tiennent lieu de fonction d'annulation, puisqu'il n'y en a pas.

**Soundboard.** Un mode par page (activé depuis le menu PAGE) qui permet à une personne qui visite de toucher des tuiles vidéo pour jouer des extraits, les extraits auto-hébergés se mixant à travers un moteur audio partagé pour que plusieurs puissent retentir en même temps.

**Trou de ver.** Dans Telaris, la plus petite unité de contenu. Une page Hotglue peut être rattachée à un trou de ver pour que celui-ci affiche ta mise en page en forme libre au lieu de son média classique.

**Galaxie.** Dans Telaris, un regroupement de trous de ver. Ton accès en modification à une galaxie (lecture-écriture, lecture seule, ou aucun) détermine quelles pages Hotglue de ses trous de ver tu peux changer.

**Média classique.** La mise en page de média simple et empilée d'un trou de ver, l'alternative à une page Hotglue. Un trou de ver affiche celui des deux (Classique ou Hotglue) qui est sélectionné au moment de son enregistrement.

## Non disponible dans cette version

La version Telaris de Hotglue laisse intentionnellement de côté les fonctionnalités qui te permettraient de coller du HTML ou des scripts arbitraires (les modules « code » et « user code »). Si tu as utilisé une autre copie de Hotglue qui les avait, voilà la différence. Tout ce que tu ajoutes ici est l'un des types de contenu dans *Ajouter du contenu*.
