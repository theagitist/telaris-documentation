# Keyboard shortcuts

Every shortcut in the Hotglue editor, in one place. These work in edit mode.

## Menus

| Shortcut | What it does |
| --- | --- |
| Single-click background | Open the NEW menu (add content) |
| <kbd>alt</kbd> + <kbd>o</kbd> | Open the NEW menu |
| Double-click background | Open the PAGE menu (whole-page tools) |
| <kbd>alt</kbd> + <kbd>p</kbd> | Open the PAGE menu |

## Selecting

| Shortcut | What it does |
| --- | --- |
| Click | Select an object |
| <kbd>shift</kbd> + click | Add or remove an object from the selection |
| Click background | Deselect everything |
| <kbd>ctrl</kbd> + <kbd>A</kbd> | Select all objects |
| <kbd>ctrl</kbd> + <kbd>D</kbd> | Select none |
| <kbd>ctrl</kbd> + <kbd>I</kbd> | Invert the selection |
| <kbd>tab</kbd> | Step to the next object |

## Moving and resizing

| Shortcut | What it does |
| --- | --- |
| Drag | Move the selection |
| <kbd>shift</kbd> + drag | Move along one axis only |
| <kbd>ctrl</kbd> + drag | Move ignoring the snap grid |
| Arrow keys | Nudge the selection one pixel |
| <kbd>shift</kbd> + arrow | Move by one grid step |
| Drag the resize handle | Resize the object |
| <kbd>ctrl</kbd> + resize | Resize ignoring the snap grid |

## Arranging and removing

| Shortcut | What it does |
| --- | --- |
| <kbd>shift</kbd> + <kbd>page up</kbd> | Bring the selection to the front |
| <kbd>shift</kbd> + <kbd>page down</kbd> | Send the selection to the back |
| <kbd>delete</kbd> | Delete the selection |

## History

| Shortcut | What it does |
| --- | --- |
| <kbd>ctrl</kbd> + <kbd>z</kbd> | Offer to open the page's revisions (there is no plain undo) |

> [!note] On Windows and Linux use Ctrl; on a Mac the same keys are labelled Control. Hotglue's shortcuts are the same across platforms.
