# Manuel d'administration

Rédigé. Les chapitres `01`..`10` sont la source ; `meta.yaml` fixe la couverture et la version. Construis avec `python3 build.py admin-manual-fr`. Couvre la Console d'administration dans l'application (Comptes, inscription autonome, accès des personnes qui éditent, galaxies/amas, paramètres globaux, fédération, sauvegardes/instantanés, maintenance) à partir de Telaris v6.12.28. Les captures d'écran référencées sous `assets/images/admin-manual-fr/` sont produites par le pipeline de captures.
